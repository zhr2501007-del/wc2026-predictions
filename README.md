1. **先核对**：读本文件，找出所有「待核」行；web 检索确认真实终场比分与晋级方，回填 `真实比分/真实晋级/胜负命中/比分命中`，重算上面「命中率汇总」。查不到就保持「待核」，下次再补。

3. **提交**：git add + commit + push。

## 结算规则
- 胜负命中：预测晋级方 == 实际晋级方（加时/点球决出的以最终晋级方为准）。
- 比分命中：仅比 **90分钟常规时间**比分，完全一致才算命中（加时进球不计入比分核对）。
